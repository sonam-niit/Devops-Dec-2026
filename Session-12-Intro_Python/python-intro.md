# Introduction to Python

- Programming langugage

- How to install Python?

[Download Python From Here](https://www.python.org/downloads/)

- If you want to install directy in Windows- just download the installer
    - double click on the same
    - continue with default option next next
    - finish installation
    - for verify open cmd: python --version

- Click on MacOs installer for Mac Users and follow the simple installation Process.
    - for verify open terminal: python --version

### For linux Installation

```bash
sudo apt update && sudo apt install python3 -y
python3 --version
```
