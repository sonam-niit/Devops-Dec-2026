# Linux Setup in Windows

- install WSL (Windows Sub System for Linux)
- just open Command Prompt (CMD)
- type wsl --install
- will setup ubuntu in your windows system
- it will ask you to setup a password
- you can setup simple password which you can remeber like admin

- Incase If you don't want to setup this you can use AWS ubuntu or linux instance but its to setup because free instances will not provide that much of features to install many packages etc..
