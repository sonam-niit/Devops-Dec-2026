# Ansible Tower

- Web Based GUI and centralized management system for ansible
- Helps:
    + run plaubooks from UI
    + schedule automated Job
    + manage inventories
    + control user access
    + store credentials
    + monitor job execution

- Ansible Tower is old name new name is AAP (ansible automation platform)
