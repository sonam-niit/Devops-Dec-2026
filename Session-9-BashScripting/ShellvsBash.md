# What is Shell?

- Shell is a program that takes command from user and gives them to OS to execute.
- It is a CLI :: Clommand Line Interpreter
- Examples:
    1. sh -- Bourne Shell (Original Unix Shell)
    2. Bash -- linux default shell
    3. zsh, ksh etc.. - mac os
    4. power shell: windows shell
- Shell is the environment in which your commands/scripts executes.

## Bash Scripting

- It is a simple file containing sequence of commands
- those commands executes inside shell step by step
- It starts with she bang (#!/bin/bash) 
- In case if you have multiple shell available I am saying use /bash as my default shell to interpret my commands.
- save the script with .sh extention
- It usually helps you to automate shell commands.
