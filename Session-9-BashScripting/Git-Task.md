# How to Colloborate With Team in Git

- Create 3-4 People Team
- from that 1 person will create Repository

- A person created a Repository
- All other B,C,D will clone the repo.
- Now A will give Colloboration invite to B,C,D

- Every developer will get invite req email, confirm the same

- Now All Developer will create their own branch in cloned repo
- Do you work
- stage it 
- commmit the same
- and push to your branch: git push origin <branch_name>

- From Git Repo you can see option to send a Pull request
- Mention your updates and ask lead developer to verify and merge
