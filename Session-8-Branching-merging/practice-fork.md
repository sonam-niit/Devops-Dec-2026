## Fork and Pull Request

- Fork my repo by clicking on form button
[Fork Repo](https://github.com/sonam-niit/first-sample-repo)

- once you fork it will make a copy of my repo to your Github account
- that means you can see same repo in your github

- now clone your repo which is created from forking
- make some changes
- Stage them
- Commit them
- push them

- then go to your repo
- you can see the button contribute
- click and there you can see create pull request
- write message like Updated XYZ
- write description like what you contributed
- create pull request

- This request will be present in my account which i can merge
- Normally Collab happen like this
